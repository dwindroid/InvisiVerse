// Technology Matrix

How We Render the Invisible

01
A-Frame WebVR
Mozilla's declarative HTML VR framework — run immersive scenes in-browser with zero plugins, full headset support via WebXR.
02
Quantum Simulation
Schrödinger equation solved numerically per-frame. Probability density rendered as animated volumetric fog and particle clouds.
03
Bioluminescence Engine
GPU-instanced micro-organism swarms with flocking behaviour (Boids) and real-time bioluminescent bloom shaders.
04
Molecular AR Overlay
WebAR marker tracking projects bond structures onto real lab benches. Electron-density isosurfaces rendered at 60fps via WebGL.
05
Field Line Tracer
Real-time vector field integration traces electromagnetic flux lines. Coulomb / Biot-Savart computed on-device via WebGPU compute shaders.
06
Haptic Feedback API
Paired VR controllers pulse resistive force when colliding with simulated force-field barriers — making the invisible tactile.
